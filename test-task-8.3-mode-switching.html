Test Task 8.3 - Mode Switching Logic

Validation de la logique de changement de mode entre upload et caméra

Test 1: Passage du mode Upload au mode Caméra
Vérifier que le passage du mode upload au mode caméra fonctionne sans perdre les données du formulaire.

Étapes du test:

  1. Ouvrir l'application et cliquer sur "Ajouter un élève"
  2. Remplir quelques champs du formulaire (nom, prénom, classe, etc.)
  3. Vérifier que le mode "Télécharger" est sélectionné par défaut
  4. Cliquer sur le bouton "Prendre une photo"
  5. Observer le changement de mode

Résultat attendu:

  • ✓ Le mode passe de "Télécharger" à "Prendre une photo"
  • ✓ L'interface de la caméra s'affiche (demande de permission puis flux vidéo)
  • ✓ Tous les champs du formulaire conservent leurs valeurs
  • ✓ La photo existante (si présente) est préservée dans formData.photoUrl
  • ✓ Le bouton "Prendre une photo" est maintenant en surbrillance (vert)
Test 2: Passage du mode Caméra au mode Upload (Test Critique)
Vérifier que le passage du mode caméra au mode upload arrête correctement la caméra et préserve les données.

Étapes du test:

  1. Ouvrir l'application et cliquer sur "Ajouter un élève"
  2. Remplir quelques champs du formulaire
  3. Cliquer sur "Prendre une photo" pour activer le mode caméra
  4. Autoriser l'accès à la caméra si demandé
  5. Attendre que le flux vidéo s'affiche
  6. Ouvrir la console du navigateur (F12) pour observer les logs
  7. Cliquer sur le bouton "Télécharger" pour revenir au mode upload
  8. Observer la console et l'interface

Résultat attendu:

  • ✓ Le mode passe de "Prendre une photo" à "Télécharger"
  • ✓ Le flux vidéo disparaît immédiatement
  • ✓ La caméra s'arrête (le voyant de la caméra s'éteint sur l'appareil)
  • ✓ L'interface de téléchargement de fichier s'affiche
  • ✓ Tous les champs du formulaire conservent leurs valeurs
  • ✓ La photo existante (si présente) est préservée
  • ✓ Aucune erreur dans la console

⚠️ Points critiques à vérifier:

  • Arrêt de la caméra: Le voyant de la caméra doit s'éteindre immédiatement
  • Libération des ressources: Vérifier dans la console qu'il n'y a pas de warning sur les tracks
  • Préservation des données: Aucune donnée du formulaire ne doit être perdue

  • ✓ La caméra redémarre à chaque passage au mode caméra
